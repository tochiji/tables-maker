# Table-maker
ダッシュボードを作成する際に必要な、DB側のテーブル設計を行うためのWebツールです。

## 概要
プロジェクト単位でテーブルとそのカラムを管理できるようにする。

プロジェクト  
　┗各テーブル  
　　┗各カラム  

カラムとして下記の要素を想定。
- カラム名
- 論理名
- 型（int、double、date、string等）
- 計算ロジック
- PKか否か
- nullを許容するかいなか
- デフォルト値
- データ元
- 例  

を入力できるようにする。

## データ保存、共有
クライアント-代理店間で保存したテーブル及びカラムをURLベースで共有し、互いに更新やコメントできるようにする必要がある。  
保存箇所はまだ未定。
